## Features Delivered

### A. Cost Attribution by Runner Type
- Table showing per-runner breakdown: Ubuntu, macOS, Windows
- Columns: Runner, Minutes, Rate ($/min), Est. Cost, Share (%)
- Visual share bars per runner
- Total row with aggregate cost

### B. Monthly Burn Rate & Projections
- 4 summary stat cards: Est. Total Cost, Total Minutes, Daily Burn Rate, Projected EOM
- Dual progress bars: time elapsed vs minutes usage
- Status indicators: ok (green), warning (amber), critical (red)
- Projection details: current usage, projected EOM, projected overage, overage cost

### C. Org Selector
- Input field to specify organization login
- Works for both org and personal billing (with deprecation handling)
- Enter key support for quick loading

### D. Pricing Reference
- Inline documentation of GitHub's published rates
- Link to official GitHub pricing docs
- Caveat about larger runners not reflected in estimates

## API Endpoint

```
GET /api/github/billing/cost-analysis?org=<org_login>
```

**Response shape:**
```typescript
{
  cost: {
    total_cost: number,        // USD
    breakdown: RunnerCostBreakdown[],
    total_minutes: number,
    included_minutes: number,
    paid_minutes: number,
    estimated: true
  },
  burn_rate: {
    days_elapsed: number,
    days_total: number,
    progress: number,          // 0-1
    current_minutes: number,
    projected_minutes: number,
    projected_overage: number,
    projected_overage_cost: number,
    daily_burn_rate: number,
    status: "ok" | "warning" | "critical"
  },
  billing: {
    kind: "user" | "org",
    login: string,
    total_minutes_used: number,
    total_paid_minutes_used: number,
    included_minutes: number,
    minutes_used_breakdown: Record<string, number>
  }
}
```

## Pricing Rates Used

| Runner | Rate ($/min) | Source |
|--------|-------------|--------|
| Ubuntu (standard) | $0.008 | GitHub docs |
| macOS (standard) | $0.08 | GitHub docs |
| Windows (standard) | $0.016 | GitHub docs |

**Caveats documented in UI:**
- Larger runners (4-core, 8-core, GPU) have higher rates not reflected
- Costs are for paid overage minutes only — included quota is free
- Burn rate projection uses Ubuntu rate as conservative baseline for future runs
