## API Endpoint

```
GET /api/github/org-overview?org=<org_login>&limit=<number>
```

- `org` (required): Organization login name
- `limit` (optional, default 20, max 50): Number of repos to fetch summaries for

**Cache TTL:** 15 minutes (this is an expensive multi-request call)

**Response shape:**
```typescript
{
  org: string,
  total_repos: number,
  active_repos: number,
  aggregate: {
    total_runs: number,
    avg_success_rate: number,
  },
  repos: [{
    repo: Repo,
    summary: RepoSummary,
    workflow_count: number,
  }]
}
```

- Each org overview request fetches: 1 (list repos) + N * 2 (workflows + summary per repo)
- Default limit=20 â†’ ~41 API calls per request
- 15-minute cache TTL mitigates repeat calls
- `Promise.allSettled` ensures partial failures don't break the page
